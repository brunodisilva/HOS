HELP

Access Name is the name of protected folder.

Directory Path is the place where files are protected, the folder "members"  contains all member protected areas, user can access to one or more protected areas upon privileges. 

For example: To add protected areas  "product1","product2", "product3" to /httpd/htdocs/members, the directory path will be:
/httpd/htdocs/members/product1
/httpd/htdocs/members/product2
/httpd/htdocs/members/product3

Access Description is the text which will be shown on member index page.

Apply new protected folder to existing member: Admin can choose if this new created folder will be added to existing active members account. Note: This selection doesn't apply to inactive account and registration account.
